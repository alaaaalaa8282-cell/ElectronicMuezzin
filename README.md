# المؤذن الإلكتروني 🕌

تطبيق إسلامي شامل لمواقيت الصلاة والأذان والأذكار

## المميزات

- **مواقيت الصلاة** - حساب دقيق بدون إنترنت بناءً على الموقع الجغرافي
- **شاشة الأذان** - شاشة كاملة عند دخول وقت الصلاة مع صوت الأذان
- **اتجاه القبلة** - بوصلة للقبلة مع المسافة للمكة المكرمة
- **الأذكار اليومية** - صباح ومساء وبعد الصلاة وأذكار متنوعة
- **التقويم الهجري والميلادي** - في شاشة واحدة مع المناسبات الإسلامية
- **شهر رمضان** - تنبيه الإفطار والسحور
- **ختم القرآن** - تتبع ختمتك وتنظيم وردك اليومي
- **عداد التسبيح** - عداد تفاعلي مع اهتزاز
- **دعم اللغة العربية** - واجهة عربية كاملة

## البناء

### متطلبات
- Android SDK 35
- JDK 17
- Gradle 8.9

### ملفات يجب إضافتها يدوياً قبل البناء

#### الخطوط (app/src/main/res/font/)
- `arabic_regular.ttf` - يمكن تحميله من [Noto Sans Arabic](https://fonts.google.com/noto/specimen/Noto+Sans+Arabic)
- `arabic_bold.ttf`

#### ملفات الأذان (app/src/main/res/raw/)
- `azan_makkah.mp3`
- `azan_fajr.mp3`
- `azan_madinah.mp3`
- `azan_egypt.mp3`
- `azan_short.mp3`

### البناء عبر GitHub Actions
بعد رفع الملفات، سيبني Actions الـ APK تلقائياً.
