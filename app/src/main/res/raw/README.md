# ملفات الأذان الصوتية

ضع هنا ملفات الأذان بصيغة .mp3 أو .ogg:

- azan_makkah.mp3    → أذان مكة المكرمة
- azan_fajr.mp3      → أذان الفجر (الصلاة خير من النوم)
- azan_madinah.mp3   → أذان المدينة المنورة
- azan_egypt.mp3     → أذان مصري
- azan_short.mp3     → أذان قصير

يمكن تحميل ملفات الأذان من المواقع الإسلامية المجانية مثل:
- https://www.islamicfinder.org
